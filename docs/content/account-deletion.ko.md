## 계정 삭제 방법

### 방법 1: 앱 내에서 직접 삭제 (권장)

1. Upvy 앱을 실행합니다
2. 하단 탭에서 **프로필** 아이콘을 탭합니다
3. 우측 상단의 **설정** (톱니바퀴 아이콘)을 탭합니다
4. 화면 하단까지 스크롤합니다
5. 로그아웃 버튼 아래의 **회원 탈퇴**를 탭합니다
6. 확인 팝업에서 **탈퇴**를 탭하면 계정이 삭제됩니다

### 방법 2: 이메일로 삭제 요청

앱에 접근할 수 없는 경우, 아래 이메일로 삭제를 요청할 수 있습니다:

- **이메일**: app.grow.snap@gmail.com
- **제목**: [계정 삭제 요청] {가입한 이메일 주소}
- **본문**: 가입 시 사용한 이메일 주소와 삭제 요청 사유를 작성해 주세요

이메일 요청 시 본인 확인을 위해 추가 정보를 요청할 수 있으며, 확인 후 7일 이내에 처리됩니다.

## 삭제되는 데이터

계정 삭제 시 다음 데이터가 **즉시 영구 삭제**됩니다:

| 데이터 유형 | 설명 |
|-----------|------|
