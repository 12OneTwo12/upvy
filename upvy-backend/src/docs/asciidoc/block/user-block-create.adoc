==== Example

[source,bash]
----
curl -X POST http://localhost:8080/api/v1/users/{userId}/block \
  -H "Authorization: Bearer {access_token}"
----

==== Notes

* 자기 자신을 차단할 수 없습니다
* 이미 차단한 사용자는 다시 차단할 수 없습니다
* 차단된 사용자의 콘텐츠는 피드, 검색 등 모든 조회에서 제외됩니다
* 차단 성공 시 201 Created를 반환합니다
