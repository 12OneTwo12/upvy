=== 차단한 콘텐츠 목록 조회

로그인한 사용자가 차단한 콘텐츠 목록을 조회합니다.

include::{snippets}/content-blocks-list/query-parameters.adoc[]

==== Response

include::{snippets}/content-blocks-list/http-response.adoc[]

==== Example

[source,bash]
----
curl -X GET http://localhost:8080/api/v1/contents/blocks?cursor={blockId}&limit=20 \
  -H "Authorization: Bearer {access_token}"
----

==== Notes

* 인증이 필요합니다
* 커서 기반 페이지네이션을 지원합니다 (block_id 기준)
* 차단한 콘텐츠가 없는 경우 빈 배열을 반환합니다
* Soft Delete된 차단은 제외됩니다
