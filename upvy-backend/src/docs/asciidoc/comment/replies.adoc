==== Example

.첫 페이지 조회
[source,bash]
----
curl -X GET "http://localhost:8080/api/v1/comments/{commentId}/replies?limit=20"
----

.다음 페이지 조회 (cursor 사용)
[source,bash]
----
curl -X GET "http://localhost:8080/api/v1/comments/{commentId}/replies?cursor={replyId}&limit=20"
----

==== Notes

* 대댓글의 대댓글은 지원하지 않습니다 (replyCount는 항상 0)
* 커서 기반 페이징으로 무한 스크롤을 구현할 수 있습니다
* `hasNext`가 `true`이면 다음 페이지가 존재합니다
* `nextCursor`를 사용하여 다음 페이지를 조회할 수 있습니다
* 삭제된 대댓글은 목록에 포함되지 않습니다
* 생성 시간 오름차순으로 정렬됩니다
* "더보기" 버튼 클릭 시 이 API를 호출하여 대댓글을 로드합니다
