==== Example

[source,bash]
----
# 기본 검색 기록 조회 (10개)
curl -X GET "http://localhost:8080/api/v1/search/history" \
  -H "Authorization: Bearer {access_token}"

# 검색 기록 개수 조정
curl -X GET "http://localhost:8080/api/v1/search/history?limit=20" \
  -H "Authorization: Bearer {access_token}"
----

==== Notes

* **인증 필수**: 로그인한 사용자만 자신의 검색 기록을 조회할 수 있습니다
* 동일한 키워드는 가장 최근 검색만 반환됩니다 (중복 제거)
* 최신순으로 정렬되어 반환됩니다
* 검색 타입 (CONTENT, USER)도 함께 반환됩니다
* 기본 제한은 10개이며, 최대 100개까지 조회 가능합니다
* 소프트 삭제된 검색 기록은 조회되지 않습니다
* 비인증 사용자의 검색은 기록되지 않습니다
