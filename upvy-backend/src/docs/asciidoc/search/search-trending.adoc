==== Example

[source,bash]
----
# 기본 인기 검색어 조회 (10개)
curl -X GET "http://localhost:8080/api/v1/search/trending"

# 인기 검색어 개수 조정
curl -X GET "http://localhost:8080/api/v1/search/trending?limit=20"
----

==== Notes

* 인증 없이도 사용 가능합니다 (공개 API)
* Redis를 사용하여 실시간 검색 횟수를 집계합니다
* 최근 24시간 동안의 검색 통계를 기준으로 순위를 결정합니다
* 순위는 1부터 시작하며, 검색 횟수가 많을수록 높은 순위를 가집니다
* 기본 제한은 10개이며, 최대 100개까지 조회 가능합니다
* 매시간 자동으로 순위가 갱신됩니다
