=== 퀴즈 통계 조회

퀴즈의 전체 통계를 조회합니다. 모든 사용자의 시도 기록과 각 보기별 선택 통계를 포함합니다.

==== Example

[source,bash]
----
curl -X GET http://localhost:8080/api/v1/quizzes/{quizId}/stats
----

==== Notes

* 인증 없이도 조회 가능합니다 (공개 통계)
* 모든 보기의 정답 여부가 공개됩니다
* `totalAttempts`: 전체 시도 횟수 (한 사용자가 여러 번 시도 가능)
* `uniqueUsers`: 참여한 고유 사용자 수
* `selectionPercentage`: 전체 시도 대비 해당 보기 선택 비율
