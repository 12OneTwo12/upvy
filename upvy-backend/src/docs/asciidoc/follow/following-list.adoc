==== Example

[source,bash]
----
curl -X GET http://localhost:8080/api/v1/follows/following/{userId} \
  -H "Authorization: Bearer {access_token}"
----

==== Notes

* 인증이 필요합니다
* 팔로잉이 없는 경우 빈 배열을 반환합니다
* Soft Delete된 팔로우 관계는 제외됩니다
