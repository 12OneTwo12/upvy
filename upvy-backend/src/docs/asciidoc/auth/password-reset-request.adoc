=== 비밀번호 재설정 요청

비밀번호를 잊어버린 경우 이메일로 인증 코드를 받아 재설정을 시작합니다.

==== Example

[source,bash]
----
curl -X POST http://localhost:8080/api/v1/auth/password/reset/request \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "language": "ko"
  }'
----

==== Notes

* 인증 불필요 (public 엔드포인트)
* 이메일로 6자리 인증 코드가 발송됩니다
* 인증 코드는 5분간 유효합니다
* 1분에 1회만 재전송 가능합니다 (rate limiting)
* 존재하지 않는 이메일이어도 보안상 204 No Content를 반환합니다
* OAuth 전용 사용자(이메일 인증 방식이 없는 사용자)에게도 204를 반환하지만 이메일은 발송하지 않습니다
* language 값: "ko" (한국어), "en" (영어), "ja" (일본어)
* 성공 시 204 No Content를 반환합니다
