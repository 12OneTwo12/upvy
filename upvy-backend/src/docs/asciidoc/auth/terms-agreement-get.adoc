=== UGC 약관 동의 조회

사용자의 현재 약관 동의 상태를 조회합니다.

==== Request

include::{snippets}/auth/terms-agreement-get/http-request.adoc[]

==== Response

include::{snippets}/auth/terms-agreement-get/http-response.adoc[]
include::{snippets}/auth/terms-agreement-get/response-fields.adoc[]

==== Example

[source,bash]
----
curl -X GET http://localhost:8080/api/v1/auth/terms-agreement \
  -H "Authorization: Bearer {access_token}"
----

==== Notes

* 인증된 사용자만 호출 가능합니다 (JWT Access Token 필요)
* 약관 동의가 존재하지 않는 경우, 모든 필드가 false/null인 기본값을 반환합니다
* `isAllRequiredAgreed` 필드로 모든 필수 약관 동의 여부를 확인할 수 있습니다
* 프로필 생성 전 약관 동의 여부를 확인하는 용도로 사용됩니다
