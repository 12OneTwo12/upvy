==== Example

[source,bash]
----
curl -X POST http://localhost:8080/api/v1/auth/password/reset/confirm \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "code": "123456",
    "newPassword": "NewSecurePassword123!"
  }'
----

==== Notes

* 인증 불필요 (public 엔드포인트)
* 프론트엔드에서 새 비밀번호 입력 후 "완료" 버튼 클릭 시 사용합니다
* 인증 코드를 다시 검증하고 비밀번호를 변경합니다
* 새 비밀번호는 최소 8자 이상이어야 합니다
* 코드가 유효하면 비밀번호를 변경하고 204 No Content를 반환합니다
* 코드가 유효하지 않으면 400 Bad Request를 반환합니다
* 코드가 만료되었으면 400 Bad Request를 반환합니다
* OAuth 전용 사용자는 403 Forbidden을 반환합니다
* 비밀번호 변경 후 사용된 인증 코드는 soft delete됩니다
* 변경 완료 후 사용자는 새 비밀번호로 로그인해야 합니다
* 기존 JWT 토큰은 유효하게 유지됩니다 (세션 무효화하지 않음)
