=== 비밀번호 변경

현재 비밀번호를 알고 있는 경우 새 비밀번호로 변경합니다.

==== Example

[source,bash]
----
curl -X POST http://localhost:8080/api/v1/auth/password/change \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer {ACCESS_TOKEN}" \
  -d '{
    "currentPassword": "OldPassword123!",
    "newPassword": "NewPassword456!"
  }'
----

==== Notes

* 인증된 사용자만 사용 가능합니다 (JWT Access Token 필요)
* 현재 비밀번호와 새 비밀번호를 모두 입력해야 합니다
* 새 비밀번호는 최소 8자 이상이어야 합니다
* OAuth 전용 사용자(이메일 인증 방식이 없는 사용자)는 사용할 수 없습니다
* 현재 비밀번호가 올바르지 않으면 401 Unauthorized를 반환합니다
* OAuth 전용 사용자가 요청하면 403 Forbidden을 반환합니다
* 비밀번호 변경 성공 시 204 No Content를 반환합니다
* 기존 세션(JWT 토큰)은 유효하게 유지됩니다
