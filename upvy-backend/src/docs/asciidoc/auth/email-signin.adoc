==== Example

[source,bash]
----
curl -X POST http://localhost:8080/api/v1/auth/email/signin \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "SecurePassword123!"
  }'
----

==== Notes

* 로그인 성공 시 JWT Access Token과 Refresh Token을 반환합니다
* Access Token은 1시간 동안 유효합니다
* Refresh Token은 14일 동안 유효합니다
* 이메일 인증이 완료되지 않은 계정은 로그인할 수 없습니다
* 잘못된 이메일 또는 비밀번호 입력 시 401 Unauthorized를 반환합니다
* 이메일 미인증 계정으로 로그인 시도 시 403 Forbidden을 반환합니다
* Access Token은 Authorization 헤더에 "Bearer {token}" 형식으로 사용합니다
* Refresh Token이 만료되기 전에 갱신하여 세션을 유지할 수 있습니다
