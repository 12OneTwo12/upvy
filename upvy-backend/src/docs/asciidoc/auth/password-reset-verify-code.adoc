==== Example

[source,bash]
----
curl -X POST http://localhost:8080/api/v1/auth/password/reset/verify-code \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "code": "123456"
  }'
----

==== Notes

* 인증 불필요 (public 엔드포인트)
* 프론트엔드에서 인증 코드 입력 후 "다음" 버튼 클릭 시 사용합니다
* 코드 검증만 수행하며, 비밀번호는 변경하지 않습니다
* 코드가 유효하면 204 No Content를 반환합니다
* 코드가 유효하지 않으면 400 Bad Request를 반환합니다
* 코드가 만료되었으면 400 Bad Request를 반환합니다
* OAuth 전용 사용자는 403 Forbidden을 반환합니다
* 검증 성공 후 다음 단계(비밀번호 변경)로 진행해야 합니다
* 같은 코드로 여러 번 검증 가능합니다 (읽기 전용 작업)
