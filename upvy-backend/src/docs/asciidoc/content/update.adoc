=== 콘텐츠 수정

콘텐츠의 메타데이터 및 사진 목록을 수정합니다. (부분 업데이트)

==== Example

[source,bash]
----
curl -X PATCH http://localhost:8080/api/v1/contents/{contentId} \
  -H "Authorization: Bearer {access_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Updated Title",
    "description": "Updated Description",
    "category": "HEALTH",
    "photoUrls": [
      "https://s3.amazonaws.com/photo1.jpg",
      "https://s3.amazonaws.com/photo2.jpg"
    ]
  }'
----

==== Notes

* 모든 필드는 선택적(optional)입니다
* 제공된 필드만 업데이트됩니다
* `photoUrls`를 제공하면 기존 사진이 모두 삭제되고 새 사진으로 교체됩니다
* `photoUrls`를 `null`로 두면 사진 목록은 변경되지 않습니다
* PHOTO 타입 콘텐츠만 `photoUrls` 필드를 사용할 수 있습니다
* 사진은 최소 1장, 최대 10장까지 업로드 가능합니다
* 본인이 작성한 콘텐츠만 수정할 수 있습니다 (403 Forbidden)
* 존재하지 않는 콘텐츠 수정 시 404 Not Found 반환
