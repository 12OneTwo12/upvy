==== Example

[source,bash]
----
curl -X DELETE http://localhost:8080/api/v1/contents/{contentId} \
  -H "Authorization: Bearer {access_token}"
----

==== Notes

* 본인이 작성한 콘텐츠만 삭제할 수 있습니다 (403 Forbidden)
* 존재하지 않는 콘텐츠 삭제 시 404 Not Found 반환
* 실제로는 Soft Delete 방식으로 처리됩니다 (데이터는 남아있음)
* 삭제된 콘텐츠는 조회 및 수정이 불가능합니다
* 삭제 성공 시 204 No Content를 반환하며 응답 본문은 없습니다
