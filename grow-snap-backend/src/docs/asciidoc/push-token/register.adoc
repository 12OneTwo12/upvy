=== 푸시 토큰 등록

푸시 알림을 위한 디바이스 토큰을 등록합니다. 동일한 deviceId로 이미 등록된 토큰이 있으면 업데이트됩니다.

==== Example

[source,bash]
----
curl -X POST http://localhost:8080/api/v1/push-tokens \
  -H "Authorization: Bearer {access_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "token": "ExponentPushToken[xxxxxxxxxxxxxx]",
    "deviceId": "device-unique-id",
    "deviceType": "IOS",
    "provider": "EXPO"
  }'
----
