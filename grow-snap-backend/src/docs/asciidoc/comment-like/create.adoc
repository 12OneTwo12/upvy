==== Example

[source,bash]
----
curl -X POST http://localhost:8080/api/v1/comments/{commentId}/likes \
  -H "Authorization: Bearer {access_token}"
----

==== Notes

* 이미 좋아요한 댓글은 다시 좋아요할 수 없습니다 (idempotent)
* 좋아요 성공 시 댓글의 좋아요 수가 반환됩니다
* 인증된 사용자만 좋아요를 추가할 수 있습니다
