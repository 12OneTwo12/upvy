==== Example

[source,bash]
----
curl -X DELETE http://localhost:8080/api/v1/comments/{commentId}/likes \
  -H "Authorization: Bearer {access_token}"
----

==== Notes

* 좋아요하지 않은 댓글도 취소 요청이 가능합니다 (idempotent)
* 좋아요 취소 시 댓글의 좋아요 수가 감소합니다
* 인증된 사용자만 좋아요를 취소할 수 있습니다
