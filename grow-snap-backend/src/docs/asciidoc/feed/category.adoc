[[feed-category]]
=== 카테고리별 피드 조회

특정 카테고리의 콘텐츠를 정렬 옵션에 따라 조회합니다.

인기순(POPULAR) 또는 최신순(RECENT) 정렬을 선택할 수 있으며, 비로그인 사용자도 조회 가능합니다.

==== Example

[source,bash]
----
# 프로그래밍 카테고리 - 인기순 첫 페이지 조회
curl -X GET "http://localhost:8080/api/v1/feed/categories/PROGRAMMING?sortBy=POPULAR&limit=20"

# 프로그래밍 카테고리 - 최신순 조회 (인증 필요 시)
curl -X GET "http://localhost:8080/api/v1/feed/categories/PROGRAMMING?sortBy=RECENT&limit=20" \
  -H "Authorization: Bearer {access_token}"

# 다음 페이지 조회 (cursor 사용)
curl -X GET "http://localhost:8080/api/v1/feed/categories/PROGRAMMING?sortBy=POPULAR&cursor=20&limit=20"
----

==== 정렬 옵션

[cols="1,3"]
|===
| 정렬 타입 | 설명

| **POPULAR**
| 인기순 정렬 (기본값) +
인터랙션 가중치 기반 인기도 점수로 정렬 +
`popularity_score = view_count * 1.0 + like_count * 5.0 + comment_count * 3.0 + save_count * 7.0 + share_count * 10.0`

| **RECENT**
| 최신순 정렬 +
콘텐츠 생성일자(created_at) 기준 내림차순 정렬
|===

==== Notes

* 커서 기반 페이지네이션을 사용하여 무한 스크롤을 지원합니다
* 비로그인 사용자도 조회 가능하며, 이 경우 좋아요/저장 상태는 모두 false로 반환됩니다
* 인증된 사용자의 경우, 각 콘텐츠에 대한 좋아요/저장 상태가 함께 반환됩니다
* 카테고리는 대소문자를 구분하며, 유효한 카테고리 enum 값이어야 합니다 (예: PROGRAMMING, DESIGN, HEALTH 등)
