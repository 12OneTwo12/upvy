=== 댓글 목록 조회 (페이징)

콘텐츠의 최상위 댓글 목록을 커서 기반 페이징으로 조회합니다.

==== Example

.첫 페이지 조회
[source,bash]
----
curl -X GET "http://localhost:8080/api/v1/contents/{contentId}/comments?limit=20"
----

.다음 페이지 조회 (cursor 사용)
[source,bash]
----
curl -X GET "http://localhost:8080/api/v1/contents/{contentId}/comments?cursor={commentId}&limit=20"
----

==== Notes

* 최상위 댓글만 반환됩니다 (대댓글은 별도 API로 조회)
* 각 댓글의 `replyCount` 필드로 대댓글 개수를 확인할 수 있습니다
* `replyCount > 0`이면 대댓글이 존재합니다
* 커서 기반 페이징으로 무한 스크롤을 구현할 수 있습니다
* `hasNext`가 `true`이면 다음 페이지가 존재합니다
* `nextCursor`를 사용하여 다음 페이지를 조회할 수 있습니다
* 삭제된 댓글은 목록에 포함되지 않습니다
* 인기순으로 정렬됩니다 (좋아요 수 + 대댓글 수 내림차순, 같은 점수면 오래된 순)
