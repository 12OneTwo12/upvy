==== Example

[source,bash]
----
curl -X GET http://localhost:8080/api/v1/videos/{videoId}/save/status \
  -H "Authorization: Bearer {access_token}"
----

==== Notes

* 인증된 사용자만 조회할 수 있습니다
* 사용자가 해당 콘텐츠를 저장했는지 여부를 반환합니다
* 실시간으로 현재 상태를 확인합니다
