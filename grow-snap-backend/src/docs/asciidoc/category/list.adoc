[[categories]]
=== 카테고리 목록 조회

사용 가능한 모든 카테고리 목록을 조회합니다.

OTHER 카테고리는 제외되며, 각 카테고리의 영문명, 표시 이름(한글), 설명을 포함합니다.

==== Example

[source,bash]
----
# 카테고리 목록 조회 (인증 불필요)
curl -X GET "http://localhost:8080/api/v1/categories"
----

==== 응답 예시

[source,json]
----
{
  "categories": [
    {
      "name": "PROGRAMMING",
      "displayName": "프로그래밍",
      "description": "개발, 코딩, 기술 관련 콘텐츠"
    },
    {
      "name": "DESIGN",
      "displayName": "디자인",
      "description": "UI/UX, 그래픽 디자인, 일러스트 등"
    },
    {
      "name": "HEALTH",
      "displayName": "건강",
      "description": "운동, 건강, 다이어트 관련 콘텐츠"
    }
    // ... 기타 카테고리들
  ]
}
----

==== Notes

* 인증이 필요하지 않은 공개 API입니다
* Redis 캐싱이 적용되어 빠른 응답 속도를 보장합니다 (TTL: 1시간)
* OTHER 카테고리는 시스템 내부용이므로 응답에서 제외됩니다
* 모든 카테고리는 미리 정의된 enum 값이며, 동적으로 추가되지 않습니다
* 카테고리별 피드 조회 시 `name` 필드의 값을 사용하세요
