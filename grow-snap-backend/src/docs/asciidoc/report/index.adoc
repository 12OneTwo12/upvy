== 신고 (Report)

사용자가 부적절한 콘텐츠, 댓글, 사용자를 신고하는 기능입니다.

=== 개요

* **인증 필요**: 로그인한 사용자만 신고 가능
* **중복 방지**: 동일한 사용자가 동일한 대상을 중복 신고할 수 없음
* **대상 타입**: CONTENT (콘텐츠), COMMENT (댓글), USER (사용자)
* **신고 타입**: OFF_TOPIC, SPAM, INAPPROPRIATE_CONTENT, COPYRIGHT, HARASSMENT, HATE_SPEECH, MISINFORMATION, OTHER
* **처리 상태**: PENDING (대기 중), APPROVED (승인됨), REJECTED (거부됨)
