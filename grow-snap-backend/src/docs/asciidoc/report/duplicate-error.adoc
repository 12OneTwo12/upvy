==== Example

[source,bash]
----
curl -X POST http://localhost:8080/api/v1/reports/content/{targetId} \
  -H "Authorization: Bearer {access_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "reportType": "SPAM"
  }'
----

==== Notes

* 동일한 사용자가 동일한 대상(콘텐츠, 댓글, 사용자)을 중복 신고할 수 없습니다
* HTTP 상태 코드 409 Conflict가 반환됩니다
* errorCode는 "DUPLICATE_REPORT"입니다
