[[profile-get-user-contents]]
=== 사용자의 콘텐츠 목록 조회

특정 사용자가 업로드한 콘텐츠 목록을 조회합니다.
프로필 화면의 콘텐츠 그리드에서 사용됩니다.

인증된 사용자의 경우 각 콘텐츠의 인터랙션 정보에 사용자별 상태 (isLiked, isSaved)가 포함됩니다.
비인증 사용자의 경우 기본 인터랙션 정보만 반환됩니다.

==== Example

[source,bash]
----
# 인증된 사용자의 경우
curl -X GET http://localhost:8080/api/v1/profiles/{targetUserId}/contents \
  -H "Authorization: Bearer {access_token}"

# 비인증 사용자의 경우 (isLiked, isSaved는 false로 반환)
curl -X GET http://localhost:8080/api/v1/profiles/{targetUserId}/contents
----
